    Detalhes do projeto:

    Bem, não tenho muito o que comentar acerca do site, visto que ele é um site extremamente simples. Foi construído com um pouco de JavaScript (se é que dá para considerar que foi utilizado JavaScript neste projeto), mas em sua totalidade é apenas CSS e HTML, não foi utilizado nenhum framework (como TailwindCss ou BootStrap) na construção, pois o objetivo deste site é apenas ter um local para disponibilizar os projetos de aplicações que faço com as tecnologias .Net (C#) e NodeJs (JavaScript).

    Este site também é para ser um indicativo do meu nível de conhecimentos em desenvolvimento Web, não sei o quanto irei melhorar visualmente o site conforme o tempo, mas com certeza ele irá representar o quanto de conhecimento eu possuo no front-end.

    Quais foram as dificuldades encontradas no desenvolvimento?

    Bem, eu estaria mentindo se dissesse que não enfrentei dificuldades no desenvolvimento por ser um site simples! Eu tive extrema dificuldade para configurar o CSS do site, não sei se é por conta de todo o conhecimento de desenvolvimento Web que eu possuo venha de vídeos que eu assisti, ou se é pelo fato de ser o primeiro projeto web que eu faço do zero sem ser aqueles projetinhos simples que fazemos em cursos. De qualquer forma eu sei que independente do motivo, isso significa que eu estou muito "cru" em conhecimento de CSS.

    Itens que mais me deram dor de cabeça foram as margens, tamanho de componentes, configurações de display, e o principal, posicionamento dos elementos e tags. Era para este site ter um rodapé escrito "Desenvolvido por murilo", porém eu não consegui configurar o rodapé para que ficasse fixado ao fim da página, mesmo vendo vídeos, recorrendo a fóruns comoStackOverflowe até mesmo utilizando o ChatGpt e o Copilot, eu não fui capaz de entender como fazer o que eu queria. Novamente isso mostra o quão "cru" eu estou, e o mais importante, me mostra que eu devo retroceder alguns passos para poder entender como configurar os elementos do site via CSS.

    Sobre atualizações:

    Pretendo assim que tiver conhecimentos o suficiente, adicionar um rodapé ao site. Fora isto, melhorar o design do site deixando-o visualmente mais atrativo, tanto na paleta de cores, quanto no quesito informações, configurar de forma correta a distribuição de elementos, espaçamento de itens; também adicionar uma imagem de fundo.

    Em quesito de funcionalidades, quem sabe no futuro o site não tenha opção de cadastro de contas para que apenas pessoas com um login possam visualizar e fazer download dos projetos. Talvez adicionar alguns conteúdos pagos para ajudar com custos de manutenção do site, renovação de domínio, custo de hospedagem, além de outros gastos (por enquando apenas sonhar é de graça).
